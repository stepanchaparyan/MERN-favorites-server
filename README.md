# Favorites cards page

Simple node server for favorites cards
### MERN - Mongo Express React Node

#### Used technologies

```
Node, Express, Mongo, Mangoose, Swagger
```

# Install guide
```sh
$ git clone ...
$ npm install
$ npm start
```
